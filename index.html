2. ContactService

This project is based on an assignment from CS320: Software Testing, Automation, and Quality Assurance (April 2024). It is a simple Java program that allows users to add, delete, and update contacts. I believe that by choosing this artifact, I can showcase my ability to select the right data structure and demonstrate how I write more efficient and well-structured methods.

Looking back at my code, I see several areas for improvement, including selecting the right algorithm and data structure, enhancing encapsulation, and improving validation and error handling. For this enhancement, I have switched to a HashMap to achieve O(1) time complexity instead of using an ArrayList. Additionally, I have implemented input validation and error handling to prevent invalid inputs and ensure errors are properly managed, preventing the application from crashing. To improve flexibility and scalability, I previously had four separate methods for updating a contact. Now, I have consolidated them into a single method, making the class more adaptable to changes without creating unnecessary methods.
